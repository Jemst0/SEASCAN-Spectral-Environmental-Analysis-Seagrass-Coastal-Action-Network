# SEASCAN-Spectral-Environmental-Analysis-Seagrass-Coastal-Action-Network

By integrating Sentinel-2 imagery processing, 1D-CNN classification, seagrass metric computation, and temporal change analysis into a single interface that requires no coding background, the system aims to make satellite-based seagrass monitoring operationally viable for Philippine coastal management.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
